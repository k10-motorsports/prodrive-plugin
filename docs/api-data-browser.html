AbsActive == true
ABS system intervenes during braking — pumping brakes to prevent lockup

Traction control cuts power to prevent wheelspin on corner exit

LatAccel extreme > 4.5G
Car is generating very high lateral G-force through a high-speed corner

TyreWearFL/FR/RL/RR < 0.35

ERS battery is critically low — driver must manage deployment carefully

DRS flap has opened — car has reduced drag for a straight-line speed boost

Car balance oscillating — driver working hard to keep it stable through a sequence

Very fast sequential gear changes detected — heel-toe or paddle-shift sequence
